# Roadmap

## MVP Phases

1. Foundation
2. Login and character creation
3. Offline map rendering and local movement
4. Multiplayer movement
5. Simple combat
6. Loot and inventory
7. NPC and quests

## Current Focus

- The first offline client step is now in place: authenticated world entry, local map rendering, and local tile movement
- Multiplayer sync, authoritative movement, and gameplay systems remain deferred to later phases
