# Event Contracts

This document defines placeholder event groups only. No event names, payloads, or implementations are finalized yet.

## Client to Server

- Session and account actions
- Character selection and creation
- Movement input
- Interaction requests
- Combat intent
- Chat and social actions

## Server to Client

- Session state
- Character roster
- World snapshot and world updates
- Combat results
- Loot and inventory updates
- Chat and system messages
