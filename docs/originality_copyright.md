# Originality and Copyright

## Non-Negotiable Rule

Aldrym may be heavily inspired by the feel and gameplay loop of classic Tibia-like MMORPGs, while still avoiding protected assets, data packs, proprietary files, and copied authored content.

## Do Not Copy

- Sprites, icons, UI art, tilesets, maps, or sound assets
- Lore, dialogue, quest text, authored map layouts, or authored gameplay data copied from another project
- Client or server files from Tibia, CipSoft products, OpenTibia distributions, or third-party derivatives
- Protected formulas, data packs, or proprietary content bundles

## Practical Guidance

- Build original lore, geography, visuals, quests, and progression content
- Use references for mood and structure, not for direct reproduction
- Common fantasy vocabulary and generic creature or item names are allowed when they are not copied as part of a protected data set or authored content bundle
